Tip:
Use the same browser width for all desktop shots to keep the README visually consistent.

## Capture Checklist

- Use desktop width around 1440px for primary captures.
- Use one mobile screenshot for responsiveness proof (optional but recommended).
- Capture realistic seeded data (orders, addresses, wishlist items).
- Avoid browser bookmarks/extensions in the screenshot frame.
- Keep one consistent zoom level (100%).

## Suggested Capture Flow

1. Home page hero and category blocks
2. Product listing with filters/sorting visible
3. Product detail with buy box and recommendations
4. Cart page with subtotal and progress
5. Checkout page with address/payment sections
6. Account page tabs (profile/addresses/payment)
7. Wishlist page with actions
8. Orders history page with statuses
9. Email confirmation screenshot from inbox
